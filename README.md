## Uruchomienie frontendu
1. Przejdź do katalogu frontend:
    ```sh
    cd Expense-tracker-frontend/expense-tracker
    ```

2. Zainstaluj zależności:
    ```sh
    npm install
    ```

3. Uruchom aplikację:
    ```sh
    npm start
    ```

## Uruchomienie backendu
1. Przejdź do katalogu backend:
    ```sh
    cd Expense-tracker-backend
    ```

2. Zainstaluj zależności:
    ```sh
    npm install
    ```

3. Uruchom serwer:
    ```sh
    npm start
    ```
